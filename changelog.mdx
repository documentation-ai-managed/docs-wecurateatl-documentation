## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom domains with SSL certificates
- Introduced webhook filtering for more granular event notifications
- Implemented document templates for faster content creation

## Improvements

- Enhanced search functionality with fuzzy matching and filters
- Improved API rate limits and performance optimization
- Updated user interface with better accessibility features

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Launched new API v2 with improved authentication and endpoints
- Added team collaboration features including real-time editing
- Introduced version comparison tools for documentation changes

## Breaking Changes

- Deprecated API v1 endpoints - migrate to v2 by end of year
- Changed webhook payload structure for better consistency
- Updated permission model to support granular access control

</Update>
