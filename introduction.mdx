## Getting Started

Wecurateatl is a comprehensive documentation management platform designed to help teams organize, collaborate, and publish their project documentation efficiently. It provides a centralized space where you can create, edit, and share documentation with your team and stakeholders. Whether you're building API docs, user guides, or internal knowledge bases, Wecurateatl streamlines the process with intuitive tools and powerful features.

## Core Features

Wecurateatl offers a range of features to enhance your documentation workflow. You can create structured documentation with markdown support, version control for tracking changes, and role-based access to ensure the right people have the right permissions.

<Tabs>
  <Tab title="Markdown Support" icon="file-text">
    Write documentation using standard markdown syntax with additional features like code highlighting and table support.
    
    <CodeGroup tabs="Markdown,HTML">
      ```markdown
      # Heading
      
      This is a paragraph with **bold** and *italic* text.
      
      | Feature | Description |
      |---------|-------------|
      | Code blocks | Syntax highlighting for multiple languages |
      | Tables | Easy table creation with markdown |
      ```
      ```html
      <h1>Heading</h1>
      <p>This is a paragraph with <strong>bold</strong> and <em>italic</em> text.</p>
      <table>
        <tr><th>Feature</th><th>Description</th></tr>
        <tr><td>Code blocks</td><td>Syntax highlighting</td></tr>
      </table>
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Version Control" icon="git-branch">
    Track changes over time with built-in version control. Compare versions, revert changes, and maintain a history of your documentation.
    
    <Steps>
      <Step title="Create Version" icon="plus">
        Create a new version of your documentation project.
        
        ```bash
        curl -X POST /api/projects/{id}/versions \
          -H "Authorization: Bearer {API_KEY}" \
          -d '{"name": "v1.1.0", "description": "New features added"}'
        ```
      </Step>
      <Step title="Compare Versions" icon="arrow-right">
        Compare two versions to see what changed.
      </Step>
      <Step title="Publish Version" icon="upload">
        Publish the selected version to make it live.
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Callout kind="info">Tip: Use our API to automate documentation updates from your CI/CD pipeline.</Callout>
